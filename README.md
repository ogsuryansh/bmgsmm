# BmgSmm

This project contains the frontend and backend for the BmgSmm site.

## Directory Structure

- `frontend`: React + Vite frontend application.
- `backend`: Backend directory (to be populated).

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

Currently empty.

## Development

The frontend is built with React and Vite, featuring a modern, visually stunning UI.
